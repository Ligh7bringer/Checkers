# Checkers
A *relatively* simple Checkers game. Written in Java using the Swing framework.

## Features
- **3 game modes**: play vs. (basic) AI, play vs. another player, spectate a game between 2 AI players
- **History of moves**.
- **Save/load replays**.
- **Undo/redo moves**.
- **UI options**.
